## Overview
This repository contains the implementation of a Decision Tree Classifier as part of my Machine Learning Internship at CODTECH IT Solutions. The model is built using `scikit-learn` to classify outcomes based on a chosen dataset.

## Objective
To build, evaluate, and visualize a Decision Tree model to understand how algorithmic splits are made based on feature importance.

## Dataset
The model utilizes the **Iris Dataset**, a standard benchmark in machine learning for multi-class classification, containing features like sepal length, sepal width, petal length, and petal width.

## Tech Stack
* **Language:** Python
* **Libraries:** scikit-learn, pandas, matplotlib, seaborn
* **Environment:** VS Code (Jupyter Notebook)

## Deliverables
* Data splitting and model training.
* Performance evaluation using accuracy score and classification reports.
* A high-resolution visualization of the generated Decision Tree.
